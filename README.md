# So long

This project is a small game developed during the 42 cursus in C with the mlx, where you are a guy how have to collect all Coca before exiting the map

## Installation

1. Clone the repo:
    ```bash
    git clone https://github.com/Pradene/so_long
    ```

2. Build the project:
   ```bash
   make
   ```

3. Run the game:
     ```bash
     ./so_long maps/[name]
     ```
